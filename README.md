Nextjs 알아보기

Nextjs는 React를 기반으로 하는 SSR 프레임워크이다.

## 프로젝트 기술스택

- Nextjs
- Chakra UI
- firebase

## 프로젝트 기능
- firebase를 이용한 Google 로그인
- Google 계정에 맞는 홈 제공
- 게시 글 작성 
- 익명 게시글 작성



.env 파일 필수. firebase 접속정보
```
publicApiKey=퍼블릭key
projectId=프로젝트id
FIREBASE_AUTH_HOST={프로젝트id}.firebaseapp.com
privateKey=프라이빗key
clientEmail=이메일
PORT=3000
HOST=localhost
PROTOCOL=http
```
